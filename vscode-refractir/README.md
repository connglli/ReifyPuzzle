# RefractIR Syntax Highlighting

This extension provides syntax highlighting for RefractIR (.sir) files.

## Features

- Syntax highlighting for keywords, types, identifiers, literals, and operators.
- Support for both line and block comments.
- Automatic bracket matching and closing.

## About RefractIR

RefractIR (also known as RefractIR) is a CFG-based symbolic IR designed for program synthesis, symbolic execution, and constraint generation for SMT solvers.
